Deploy Gateway on Railway

Since you purchased the Railway Hobby Plan, you can host your private WhatsApp API container inside the exact same project! Follow this simple 3-step guide:

1

Add New Service in Railway

Go to your Railway Dashboard project, click + New ➔ GitHub Repo ➔ Select your yss repository.

2

Configure Service Settings

Click on the newly created service ➔ go to Settings and set:

  • Root Directory: /whatsapp
  • Service Name: Rename it to whatsapp-gateway
3

Enable Persistent Session Storage

To ensure you don't have to scan the QR code again when the container restarts, attach a Volume:

  • Go to the service's Volume tab ➔ Click + Add Volume.
  • Mount Path: Set to /app/auth_info_baileys
4

Configure Backend Environment Variable

Go to your **Main Flask App** service on Railway ➔ **Variables** tab ➔ Click **New Variable**:

NAME: WHATSAPP_GATEWAY_URL
VALUE: http://whatsapp-gateway.railway.internal:3000
